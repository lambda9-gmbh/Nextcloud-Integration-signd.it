All notable changes to this project will be documented in this file.

## 2.0.0

### Changed

- Nextcloud 33 support (dropped NC 30–32)
- Requires PHP 8.2+

## 1.0.0

Initial release of the signd.it Integration for Nextcloud.

### Added

- Admin settings for API key management, login, and registration
- "Digitally sign" file action for PDFs with sidebar tab showing process status, signers, and actions
- Overview page with process list, filters, sorting, and detail sidebar
- PDF download of signed documents
- Translations: en, de, es, fr, it, pt, da, pl
- Supported Nextcloud versions: 30, 31, 32
